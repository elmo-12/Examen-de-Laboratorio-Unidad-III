# Guía de Usuario

## Acceso al Sistema

1. Abrir navegador web
2. Acceder a: `http://localhost:8501`
3. El sistema mostrará el dashboard principal

## Navegación

El sistema tiene las siguientes secciones principales:

### 1. Dashboard Principal
- Vista general del sistema
- Métricas clave
- Notificaciones pendientes
- Acceso rápido a funciones principales

### 2. Gestión de Equipos
- **Lista de Equipos:** Ver y filtrar equipos
- **Nuevo Equipo:** Registrar nuevo equipo en el inventario
- **Estadísticas:** Ver estadísticas de equipos

### 3. Gestión de Proveedores
- **Lista de Proveedores:** Ver y gestionar proveedores
- **Nuevo Proveedor:** Registrar nuevo proveedor
- **Contratos:** Gestionar contratos con proveedores

### 4. Gestión de Mantenimientos
- **Lista de Mantenimientos:** Ver historial de mantenimientos
- **Nuevo Mantenimiento:** Programar mantenimiento preventivo o correctivo
- **Calendario:** Ver mantenimientos programados por mes
- **Estadísticas:** Análisis de costos y frecuencia

### 5. Reportes y Análisis
- **Dashboard:** Métricas generales
- **Gráficos:** Visualizaciones estadísticas
- **Exportar:** Generar reportes en PDF o Excel
- **Análisis Avanzado:** Análisis detallados

## Funcionalidades Principales

### Registrar un Equipo
1. Ir a "Gestión de Equipos"
2. Pestaña "Nuevo Equipo"
3. Completar formulario
4. Clic en "Guardar Equipo"

### Programar Mantenimiento
1. Ir a "Gestión de Mantenimientos"
2. Pestaña "Nuevo Mantenimiento"
3. Seleccionar equipo
4. Elegir tipo (preventivo/correctivo)
5. Fecha y descripción
6. Guardar

### Ver Reportes
1. Ir a "Reportes y Análisis"
2. Seleccionar tipo de reporte
3. Configurar parámetros
4. Generar o exportar

### Ejecutar Agentes
1. En el sidebar del dashboard
2. Clic en "Ejecutar Agentes"
3. Los agentes revisarán:
   - Mantenimientos próximos
   - Equipos obsoletos
   - Garantías por vencer
   - Costos de mantenimiento

## Notificaciones

Las notificaciones aparecen en el sidebar del dashboard:
- Mantenimientos próximos
- Equipos obsoletos
- Garantías por vencer
- Alertas de costos

## Consejos de Uso

1. **Mantenimientos Preventivos:** Programar con anticipación
2. **Códigos de Inventario:** Usar formato consistente (ej: EQ-2024-001)
3. **Actualizar Estados:** Mantener estados de equipos actualizados
4. **Revisar Notificaciones:** Revisar regularmente las alertas
